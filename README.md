# SolarBat-AI Home Assistant Add-on

This add-on runs [SolarBat-AI](https://github.com/rszemeti/SolarBat-AI) as a standalone Home Assistant add-on — no AppDaemon add-on required.

## Quick Start

1. Add this repo in HA: Settings → Add-ons → Add-on Store → ⋮ → Repositories
2. Paste: `https://github.com/rszemeti/solarbat-ai-addon`
3. Install SolarBat-AI, configure `apps.yaml`, start

See [DOCS.md](solarbat-ai/DOCS.md) for full instructions.
